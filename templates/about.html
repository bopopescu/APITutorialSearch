API Era is Coming

    With the development of Internet and cloud computing, API business, and business transactions through API in terms of business functions and performance, it promotes the sustainable development of API sharing economy and the continuous improvement of API ecology.
    A new pattern based on API economy and supported by API ecosystem is affecting all industry users, IT vendors and cloud service providers. API Era is Coming.

I have Solid Idea


    With the increasing number and complexity of API, developers are facing more and more challenges in choosing API and dealing with the problems brought by API. Then, various API tutorials have been created to help developers solve API problems encountered during development.
    However, these API tutorials still have many deficiencies. Such as the lack of unified norms, knowledge fragmentation, traditional search methods are not effective and so on.
